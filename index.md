## Data Storage and Security

### Local Storage
All data is primarily stored on your device using iOS secure storage mechanisms. This data is protected by your device's security features (passcode, Face ID, Touch ID).

### iCloud Sync (Optional)
If you enable iCloud sync:
- Data is stored in **your personal iCloud account**
- Sync is handled by Apple's CloudKit/SwiftData
- We do not have access to your iCloud data
- Data is encrypted in transit and at rest by Apple
- You can disable iCloud sync anytime in iOS Settings

### No Server Storage
We do not operate servers. All data processing happens on your device.

### Data Security
- iOS built-in encryption protects local data
- iCloud uses Apple's security infrastructure
- No data transmitted to third-party servers (except weather API for weather information only)

## European Union Privacy Rights (GDPR)

2. **Right to Rectification:** Request correction of inaccurate data
3. **Right to Erasure:** Request deletion ("right to be forgotten")
4. **Right to Restriction:** Request limitation of processing
5. **Right to Data Portability:** Receive your data in a portable format
6. **Right to Object:** Object to processing of your data
7. **Right to Withdraw Consent:** Withdraw consent at any time

### Legal Basis for Processing
